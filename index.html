How to

Connect your cloud library.

Dropbox and Google Drive now follow the same in-app flow, so signing in and returning to your library is much simpler.

01

Choose your provider

  1. Open ChordLab Settings and choose Dropbox or Google Drive.
  2. Start the connection from inside the app.
  3. ChordLab opens a secure sign-in tab without sending you into a separate workflow.
02

Sign in and return to the app

  1. Approve access in Dropbox or Google.
  2. After authorization, ChordLab returns you straight back into the app.
  3. The library view refreshes so you can see the connected status without reopening the app.
03

Start browsing and refreshing

  1. The first successful connection can trigger a library refresh automatically.
  2. Later refreshes show a clear progress window, then close automatically when syncing finishes.
  3. While a cloud song is downloading, the library shows fetching progress and clear timeout or no-internet messages.

Help

Quick start

  1. Add charts. Open Library, import PDF, JPG, JPEG, or PNG files, or connect Dropbox or Google Drive.
  2. Create a setlist. Add songs to a setlist and drag the left handle to reorder.
  3. Open Live Mode. Use large navigation controls for performance.
  4. Use Band Mode. Turn Visible on, enter your musician name, then add or scan members on the same network.
  5. Send a chart. Open a song, select a remote member, and press Send in Band Mode. PDF and image charts can both be shared.

FAQ

Common questions

Do all tablets need the same Dropbox or Google account?

No. If the receiving tablet does not already have the chart, ChordLab can receive the sent file and store it locally.

Does Band Mode use the internet?

Band Mode is designed for devices on the same local network. It does not require a ChordLab cloud server.

Do all tablets need the same Dropbox or Google account?

No. If the receiving tablet does not already have the chart, ChordLab can receive the sent file and store it locally.

Does Band Mode use the internet?

Band Mode is designed for devices on the same local network. It does not require a ChordLab cloud server.

Can I use JPG or PNG charts too?

Yes. ChordLab supports PDF, JPG, JPEG, and PNG for viewing, annotation, setlists, and sending to members. Transpose is only for chord-based charts.

What if a cloud song gets stuck while downloading?

ChordLab shows fetching progress in the library, gives a no-internet message when the network drops, and times out instead of hanging forever.
