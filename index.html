What ChordLab does

Built for musicians who need charts fast.

為現場演奏、敬拜隊、Band rehearsal 而設，重點係快、清楚、可靠。

PDF

PDF song library

Import local PDFs or sync a Dropbox folder, then browse and search your charts quickly.

支援 Local PDF import 同 Dropbox 同步，方便建立自己嘅樂譜庫。

Set

Setlists for performance

Create setlists, reorder songs by dragging, and keep your live set ready for the next rehearsal.

建立 Setlist、拖放調整歌曲次序，演出前快速整理流程。

Key

Transpose chords

Move charts up or down by semitone, including common slash chords.

一鍵升降 key，支援常見 slash chords，方便配合唔同歌手。

Pen

Annotations

Use pen and eraser tools to mark charts directly during practice or performance.

可以用 pen tool 喺譜上做記號，排練同 live mode 都適合。

LAN

Band Mode sharing

Send the current chart to online band members on the same local network.

Band Mode 可以喺同一 Wi-Fi/LAN 傳送當前歌曲俾其他樂手。

Msg

Broadcast messages

Send short notes to remote members, such as “repeat chorus” or “go to bridge”.

可以發送簡短訊息，例如「repeat chorus」或者「去 bridge」。

Help

Quick start

  1. Add charts. Open Library, choose Local Import PDF or connect Dropbox.
  2. Create a setlist. Add songs to a setlist and drag the left handle to reorder.
  3. Open Live Mode. Use large navigation controls for performance.
  4. Use Band Mode. Turn Visible on, enter your musician name, then add or scan members on the same network.
  5. Send a chart. Open a song, select a remote member, and press Send in Band Mode.

使用幫助

快速開始

  1. 加入樂譜。 到 Library，選擇 Local Import PDF，或者連接 Dropbox。
  2. 建立 Setlist。 將歌曲加入 Setlist，用左邊三條線拖動排序。
  3. 開 Live Mode。 用大按鈕翻頁，適合演出時使用。
  4. 使用 Band Mode。 開啟 Visible，輸入名字，然後掃描或手動加入同網絡成員。
  5. 傳送歌曲。 打開歌曲後，在 Band Mode 選擇 remote member，按 Send。

FAQ 常見問題

Practical answers for rehearsals.

Do all tablets need the same Dropbox account?

No. If the receiving tablet does not have the PDF, ChordLab can receive the sent PDF and store it locally.

唔一定需要同一個 Dropbox。如果對方冇嗰份 PDF，ChordLab 可以接收 PDF 並儲存在本機。

Does Band Mode use the internet?

Band Mode is designed for devices on the same local network. It does not require a ChordLab cloud server.

Band Mode 主要用同一個 LAN/Wi-Fi，唔需要 ChordLab 雲端 server。

Why can a device not be found?

Check that both devices are on the same network, Visible is on, and the network allows devices to reach each other.

請確認兩部機在同一網絡、Visible 已開啟，而且 Wi-Fi 沒有啟用 client isolation。

Can I remove local PDFs?

Yes. Local library songs can be removed from the app without affecting Dropbox files.

可以。Local Library 的歌曲可以移除，而且唔會影響 Dropbox 原檔。

Privacy Policy 私隱政策

ChordLab is designed to keep your charts close to you.

Last updated: 14 May 2026

English

ChordLab is a music chart and setlist app. The app stores imported PDFs, parsed song information, setlists, annotations, and settings on your device.

If you connect Dropbox, ChordLab accesses the Dropbox folder you choose so it can read PDF files and sync them into the app. Dropbox authentication is handled through Dropbox.

Band Mode communicates with other ChordLab devices on the same local network. When you send a chart, the app may send chart metadata and, when needed, the PDF file to the receiving device.

ChordLab does not operate a separate cloud server for your songs and does not sell personal data. Network and file permissions are used only to provide the app features described here.

To request support or data removal guidance, contact the developer using the support details below.

中文

ChordLab 是一個樂譜及 Setlist 管理 App。App 會將匯入的 PDF、歌曲資料、Setlist、手寫標記及設定儲存在你的裝置上。

如果你連接 Dropbox，ChordLab 會讀取你指定的 Dropbox 資料夾，用作同步 PDF 樂譜。Dropbox 登入及授權由 Dropbox 處理。

Band Mode 會與同一個本地網絡上的 ChordLab 裝置通訊。當你傳送歌曲時，App 可能會傳送歌曲資料；如對方沒有該 PDF，亦可能會傳送 PDF 檔案到對方裝置。

ChordLab 不營運額外雲端伺服器儲存你的歌曲，亦不會出售個人資料。網絡及檔案權限只會用作提供上述功能。
