Quick start

Start simple, then scale up.

  1. Open Library and import PDF, JPG, JPEG, or PNG charts, or connect Dropbox or Google Drive.
  2. Add songs into a setlist and drag to reorder the flow.
  3. Open the chart, transpose if needed, and switch into Live Mode for larger controls.
  4. Use annotation tools to mark reminders, cues, and changes directly on the chart.
  5. Turn on Band Mode, connect members on the same network, and send charts or messages when needed.

FAQ

Answers that matter on rehearsal day.

Do all tablets need the same Dropbox or Google account?

No. If the receiving tablet does not already have the chart, Chordasy can receive the sent file and store it locally.

Does Band Mode use the internet?

Band Mode is designed for devices on the same local network. It does not require a Chordasy cloud server.

Can I use image charts too?

Yes. Chordasy supports PDF, JPG, JPEG, and PNG for viewing, annotation, setlists, and sending to members. Transpose is only for chord-based charts.
