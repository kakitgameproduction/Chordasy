How to

Connect Dropbox to ChordLab.

Follow these steps before you connect Dropbox inside the app.

01

Create your Dropbox app

  1. Go to the Dropbox App Console and sign in with your Dropbox account.
  2. Click Create app, choose Scoped access, then choose Full Dropbox or App folder.
  3. Give the app a name, open its Settings tab, then copy the App key.
02

Enable permissions

  1. Open the Permissions tab in the Dropbox App Console.
  2. Enable file and folder read permissions, including metadata read access.
  3. Save or submit the permission changes before returning to ChordLab.
03

Authorize in ChordLab

  1. Paste the App key into ChordLab Settings, then tap Start OAuth.
  2. Approve access in Dropbox and copy the authorization code.
  3. Paste the code into ChordLab, tap Finish OAuth, then Validate Connection.

Help

Quick start

  1. Add charts. Open Library, choose Local Import PDF or connect Dropbox.
  2. Create a setlist. Add songs to a setlist and drag the left handle to reorder.
  3. Open Live Mode. Use large navigation controls for performance.
  4. Use Band Mode. Turn Visible on, enter your musician name, then add or scan members on the same network.
  5. Send a chart. Open a song, select a remote member, and press Send in Band Mode.

FAQ

Common questions

Do all tablets need the same Dropbox account?

No. If the receiving tablet does not have the PDF, ChordLab can receive the sent PDF and store it locally.

Does Band Mode use the internet?

Band Mode is designed for devices on the same local network. It does not require a ChordLab cloud server.

Why can a device not be found?

Check that both devices are on the same network, Visible is on, and the network allows devices to reach each other.

Can I remove local PDFs?

Yes. Local library songs can be removed from the app without affecting Dropbox files.
